LaraPro

LaraPro es una aplicación web que permite realizar exámenes online. El usuario tendrá un tiempo determinado para responder a las preguntas tipo test y podrá enviar los resultados con una firma digital. Cuando un examen es aprobado la aplicación imprime un diploma con los datos del estudiate. La aplicación también tiene un perfil para profesores los cuales pueden publicar nuevos exámenes y ver información sobre test propuestos anteriormente.

La la aplicación esta construida en PHP usando el Framework Laravel y para la persistencia se ha hecho uso de MySQL. Las vistas son hechas con el motor de plantilla Blade y el diseño se realizó con el Framework ENYO.js.

Para hacer posible enviar documentos firmados digital-mente se ha hecho uso del servicio de Viafirma el cual hace posible la forma digital para los ciudadanos residentes en España.
