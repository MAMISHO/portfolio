Bici

Bici es una aplicación que realicé como proyecto de control de datos extraídos de APIs publicas expuestas como servicios web. La aplicación esta construida con tecnologías web y es posible embeberla con Phonegap para que pueda ser desplegada en dispositivos móviles con sistemas operativos como iOS o Android.

La aplicación permite saber el numero de bicicletas y estacionamientos libres de las mismas ubicadas en una ciudad. El funcionamiento es sencillo. La aplicación muestra una lista de países con sus respectivas ciudades, una vez seleccionada la ciudad nos aparecerán las estaciones de bicicletas publicas disponibles. Al seleccionar una estación podremos sabes los slots libres y las bicicletas disponibles.

Los datos de la API son extraídos del sitio citybik y están combinados con el servicio de mapas MapBox. Una funcionalidad extra de la aplicación es permitir ver el clima de la ciudad en la que se realizan las consultas, esto es posible gracias a la API de Open Weather.
